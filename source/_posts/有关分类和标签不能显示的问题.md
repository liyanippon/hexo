有些很小的问题，虽然不影响使用。但是对于游戏只要求的我来说，有些改动还是很有必要的

<!-- more -->

## 博文分类和标签不能很好的显示
### 默认是没有 categories 和 tags 的需要
``` bash
hexo new page "tags" 

hexo new page "categories"
```
### 还要确认主题配置文件
``` bash
  home: /
  archives: /archives/
  categories: /categories/
  tags: /tags/
  about: /about/
```
### 编辑 /tags/index.md /categories/index.md 这一点非常重要
``` bash
type: "tags"
layout: "tags"
 
 
type: "categories"
layout: "categories"
```
## 想在新建博文里默认显示更多的信息
### 修改md文件即可
``` bash 
 vim scaffolds/post.md 
```
